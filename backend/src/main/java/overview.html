OilWhere CRM Application

The OilWhere CRM is a comprehensive application designed to manage customer relationships, sales processes, and business operations effectively.

This documentation provides an overview of the packages and components of the application:

  • Config: Handles configuration for the SMTP email server.
  • Controller: Manages API requests and responses.
  • Model: Contains the core business entities.
  • DTO: Data Transfer Objects for communication between layers.
  • Repository: Manages database interactions.
  • Service: Implements business logic and workflows.
  • Util: Contains helper classes, such as the `EmailBuilderUtil`.
